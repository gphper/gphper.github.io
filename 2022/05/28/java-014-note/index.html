Collection

  • Vector、ArrayList对比

| | 底层结构 | 版本 | 线程安全、效率 | 扩容倍数 |

| ArrayList | 可变数组 |jdk1.2 | 不安全，效率高 | 无参时初始为10，然后1.5倍扩容|

set

  • 不重复存储 Hash值且equals结果 相同的对象
  • 可以通过重写 hashCode和equals 的方式实现set中属性相同的对象是否能同时存在多个，默认是可以存在多个的，因为不同的对象默认的哈希值不同 快捷方式可以使用 alt+insert 选 equals and hashCode
